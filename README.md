# analyse-detailwalmart
Analyse statistique des ventes de Walmart. Un test T prouve que les promotions augmentent les ventes de manière significative (p &lt; 0.001). La découverte clé : le fait d'activer une promo est efficace, mais le montant de la remise n'a qu'une très faible influence linéaire (Corrélation = 0.065) sur le volume total des ventes.
